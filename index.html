The Wiki-Learning Game


Welcome to the Wiki-Learning Game!


This Game was created to help you study, practise and memorize facts about various topics and have lots of fun while doing so.


How it works

  1. Select a topic:

    2. The mechanics of this game are pretty simple: To start a new game, you first have to select a topic. After you have selected a topic, you will automatically be forwarded to the first level.

  2. Find the mistakes:

    3. Each level consists of paragraphs from a Wikipedia article linked to the chosen topic, but if you browse through the contents, you will quickly realize that there are several mistakes hidden in the text.

  3. Correct the mistakes:

    4. If you believe that you have spotted a mistake, you can find out whether your observation is correct by clicking on said word. If it really is a mistake, a text box will appear in place of the word and you can then enter the word that you think is correct instead.
    If you have entered the correct word, the letters of this word will become green, if you have not yet found the correct word, the letters will remain red.

  4. Move on to the next level:

    5. Once you have corrected all the mistakes, a button which directs you to the next level will appear on the bottom of the page.

    Push the button to start a new game!
